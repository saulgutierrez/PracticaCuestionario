Javascript: Todo lo que necesita saber
¿Que es Javascript?

Es un lenguaje de programación ligero, interpretado, o compilado just-in-time con funciones de primera clase.

¿Que hace Javascript?

Se encarga de dotar de mayor interactividad y dinamismo a las páginas web. Cuando JavaScript se ejecuta en el navegador, no necesita de un compilador. El navegador lee directamente el código, sin necesidad de terceros.

¿Que es el DOM?

Es una API definida para representar e interactuar con cualquier documento HTML o XML. El DOM es una de las APIs más usadas en la Web, pues permite ejecutar código en el navegador para acceder e interactuar con cualquier nodo del documento.

¿Que es una API web?

Es una interfaz para páginas y aplicaciones web. Se utiliza en el intercambio y procesamiento de contenidos entre webs y permite a terceros acceder a grupos de datos y círculos de usuarios.

¿Que es lo que JavaScript hace en tu página web?

Agregar caracteristicas intuitivas al sitio web, adémas de crear una mayor interacción con el usuario, por lo que permite crear entornos virtuales, animaciones, juegos, secuencias de carga suaves, entre otro tipo de actividades que hacen la experiencia de usuario más placentera.

- Se puede hacer usando la etiqueta script que debe abarcar todo el código Javascript
- Importando desde un fichero externo, haciendo referencia a este archivo con una etiqueta link

¿Cómo se comenta el código en JavaScript?

Los comentarios en línea que comienzan con una doble barra, y los comentarios multilínea, que comienzan con /* y terminan con */.

¿Cómo se declara una variable y cuáles son sus principales características?

Usamos la palabra reservada var para declarar una o mas variables. Estas sirver para almacenar valores de todo tipo en la memoria, y guardar datos para ser utilizados despues.

¿Cómo se declara una condición y cuáles son sus principales características?

Utilizaremos "if else" para poder declarar la condicion dentro de estos parametros, con esto podemos obtener el resultado como un valor lógico

¿Cómo se declara bucle y cuáles son sus principales características?

Se usara la sintaxis "while (condicion)" donde dentro pondremos la condicion que hara que se pare el bucle, Cada vez que el bucle se repite se efectúa una iteración.

¿Cómo se declara una función y cuáles son sus principales características?

Una definición de función consta de la palabra clave function, seguida de el nombre de la función, una lista de parámetros de la función entre paréntesis y separados por comas.

¿Cómo se declara un objeto y cuáles son sus principales características?

La manera más simple de construir un objeto en JavaScript, es declarar una nueva variable de tipo Object y asignarle las propiedades o métodos que el objeto necesite.
En JavaScript, los objetos son básicamente tablas hash, esto es, un grupo de propiedades y funciones que pueden ser accedidos a través de una clave. Tanto las propiedades como los métodos pueden ser creados dinámicamente en tiempo de ejecución.

¿Cómo se puede leer un evento con JavaScript?

Se construye mediante el prefijo on, seguido del nombre en inglés de la acción asociada al evento. Así, el evento de pinchar un elemento con el ratón se denomina onclick y el evento asociado a la acción de mover el ratón se denomina onmousemove.

¿Qué eventos podemos leer con JavaScript menciónales y descríbalos?

- onblur: Deselecciona el elemento
- onchange: Deseleccionar un elmento que se ha modificado
- onclick: Pinchar y soltar el raton
- onfocus: Seleccionar un elemento
- onkeydown: Pulsar una tecla(sin soltar)

Menciona un ejemplo donde Javascript sea ejecutado del lado del servidor

Usando herramientas que permiten el uso de Javascript del lado del servidor, como NodeJS, es posible trabajar del lado del servidor, programando en javascript, introduciendo el concepto de callback, para permitir procesos en segundo plano, por detrás de lo que esta ocurriendo en el navegador. Un ejemplo muy claro de esto, seria el procesamiento de pagos, o confirmaciones de acciones importantes, las cuales, en muchos casos no requieren que la pagina principal se refresque para que las acciones ocurran, sino que se introduce en concepto de llamadas asincronas, interactuando con el servidor.

¿Qué tipo de errores podemos encontrar en Javascript?

- Uncaught TypeError : cannot read property
Este error suele ocurrir con mucha frecuencia, debido generalmente a una incorrecta inicialización de diversos componentes.
- TypeError: ‘undefined’ is not a function
Este error suele ocurrir con frecuencia cuando se llama a una función que no esta definida
- Uncaught TypeError: Cannot set property
Este error aparece cuando tratamos de acceder a una variable no definida

¿De que manera podemos ver los errores de Javascript en el navegador?

- En Google Chrome, pulsar las teclas Ctrl + Shift + J en una PC o Command + Shift + J desde una Mac.
- Apple Safari, seleccionar el menú Safari -> Preferencias -> Avanzado -> Mostrar menú de desarrollo en barra de menú

¿Cómo puedo convertir Strings a Objetos?

Si tenemos un string con un JSON lo podemos transformar en un objeto javaScript usando la función JSON. parse(json) que nos devolverá el objeto correspondiente a nuestro JSON.

¿Cómo se usan los arreglos en JavaScript?

En javascript un arreglo es una colección ordenada de elementos no homogéneos, cada elemento puede ser de un tipo de dato diferente. En javascript los arreglos empiezan con el subíndice 0.
Se pueden definir y poblar de forma simultanea:
var tabla = new Array(25,35,12,34);
var tabla = [25,35,12,34];
Tambien se pueden definir primero y llemnar despues:
var tabla = new Array();
var tabla = new Array(7);
var tabla = [];
var tabla = [7];
tabla[0] = "lunes";
tabla[1] = "martes";

¿Qué es WebGL?

interfaz de programación de aplicaciones gráficas (API) creada para su uso en aplicaciones web. Se basa en el lenguaje integrado de gráficos abiertos (OpenGL).

¿Que es un webSocket?

Es una tecnología avanzada que hace posible abrir una sesión de comunicación interactiva entre el navegador del usuario y un servidor. Con esta API, puede enviar mensajes a un servidor y recibir respuestas controladas por eventos sin tener que consultar al servidor para una respuesta.

¿Cómo gestiona la memoria JavaScript?

En JavaScript se reserva memoria cuando cosas, como objetos, strings, etc. son creados y "automáticamente" liberados cuando ya no son utilizados. El proceso anterior es conocido como Recolección de basura.
